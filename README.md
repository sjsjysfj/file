# Image Processor Pro (图像处理专业版)

## 简介
这是一个功能完整的桌面应用程序，专为高效处理图像而设计。核心功能包括图像四等分分割和竖向长图拼接。

## 功能特性

### 1. 图像分割模块
- **四等分分割**：自动将图片切割为左上、右上、左下、右下四张等尺寸图片。
- **批量处理**：支持拖拽多张图片或文件夹进行批量分割。
- **格式支持**：支持 JPG, PNG, BMP 等常见格式。
- **质量保护**：保留原始图片质量和元数据。

### 2. 竖图拼接模块
- **无缝拼接**：将多张竖向图片拼接为一张长图。
- **智能对齐**：
  - **Resize (统一宽度)**：自动缩放图片至统一宽度（推荐）。
  - **Crop (裁剪)**：按最小宽度裁剪。
  - **Fill (填充)**：按最大宽度填充背景。
- **顺序调整**：支持拖拽调整拼接顺序。

### 3. 用户界面
- **拖拽导入**：支持文件和文件夹拖拽。
- **任务队列**：实时显示处理进度和状态。
- **自定义输出**：支持自定义输出目录和格式转换。

## 安装与运行

### 源码运行
1. 确保已安装 Python 3.8+。
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行程序：
   ```bash
   python run.py
   ```

### 打包为 EXE
1. 安装 PyInstaller：
   ```bash
   pip install pyinstaller
   ```
2. 运行打包脚本：
   ```bash
   pyinstaller --noconfirm --onedir --windowed --name "ImageProcessorPro" --add-data "src;src" run.py
   ```
   或者直接运行提供的构建脚本（如果有）。

## 操作指南

### 图像分割
1. 切换到“图像分割”标签页。
2. 将图片拖拽到虚线框区域，或点击拖拽区域添加文件。
3. 在左侧设置面板选择输出格式（可选）和输出目录。
4. 点击“开始处理”。

### 图片拼接
1. 切换到“竖图拼接”标签页。
2. 将需要拼接的图片拖拽到列表区域。
3. 在列表中拖拽图片名称以调整拼接顺序（从上到下）。
4. 在左侧选择“拼接模式”（建议使用 Resize）。
5. 点击“开始处理”。

## 技术栈
- **UI 框架**: PyQt6
- **图像处理**: Pillow (PIL)
- **并发处理**: QThreadPool
